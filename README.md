# 📌 Project Overview

Netflix is one of the world's leading streaming platforms with thousands of movies and TV shows. This project performs Exploratory Data Analysis (EDA) on Netflix's content catalog to uncover valuable insights about genres, ratings, countries, release years, and content distribution.

The project demonstrates data cleaning, preprocessing, visualization, and business intelligence techniques using Python.

---

# 🎯 Problem Statement

Streaming platforms contain massive amounts of content, making it difficult to understand viewing trends and content distribution.

This project analyzes Netflix's dataset to answer important business questions and extract meaningful insights using data analysis techniques.

---

# 🎯 Objectives

- Perform data cleaning
- Handle missing values
- Remove duplicate records
- Analyze Movies vs TV Shows
- Analyze release year trends
- Discover top genres
- Identify top producing countries

- Visualize important trends
- Generate business insights
